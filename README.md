# Terraform

## Commande Terraform
- **init** : initialiser le terraform
- **init -upgrade** : mettre a jour l'initialisation du terraform
- **fmt** : reformater tous les fichiers de terraform (.tf)
- **validate** : vérifie la syntaxe des fichiers de terraform (.tf)
- **plan** : retourne le plan que le terraform exécutera
- **apply** : exécuter le terraform
- **destroy** : destroy toute la configuration (équivalence du bouton reset dans AWS)
- **output** : revoir les résultas des output
